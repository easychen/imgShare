imgShare
========

为文章中的图片添加鼠标浮动时分享到微博的链接。封装自 国内首家互联网人才拍卖网站 JobDeer.com

## 为什么要使用imgShare？
作为Growth Hacking的一个促进用户分享内容的技巧。详细文章：http://get.jobdeer.com/449.get


## 预览图

![img](a.png?raw=1)

## 使用方式

在页面中引用JS和CSS后，对需要分享图片的区域调用代码

``` javascript
$('selector').imgShare();
```

即可。
